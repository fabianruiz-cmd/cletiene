Este documento describe los endpoints y lineamientos técnicos necesarios para integrar sistemas externos con la plataforma Wip.

La API permite:

Crear servicios
Consultar servicios por diferentes criterios
Consultar suscripciones
Recibir actualizaciones en tiempo real mediante Webhooks
Gestionar información operativa asociada a servicios en campo

Alcance
Esta documentación está dirigida a equipos técnicos (developers, arquitectos de software y líderes de integración) responsables de implementar la conexión entre sus sistemas internos y Wip.
